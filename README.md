# 🖥️ Retro Windows 95-Style Website

A nostalgic Windows 95-themed personal website with interactive desktop elements.

## ✨ Features

- Interactive desktop with draggable icons
- Start menu with functional links
- Music player with playlist functionality
- "My Computer" explorer with contact form
- Windows 95 visual styling and interface

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in your browser
